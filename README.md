# 旅のしおりアプリ (Ryoko no Shiori)

スタイリッシュなデザインと、オフラインでも使えるような手軽な操作性を目指して開発された、iPhone向け旅のしおりWebアプリケーションです。

## 📱 プロジェクト概要

*   **ターゲット**: iPhone (モバイルファーストデザイン)
*   **デザイン**: スペイン・ポルトガルの雰囲気を意識したモダンなUI
*   **主な特徴**: 
    *   複数の旅行を個別に管理可能。
    *   すべてのデータはブラウザの `localStorage` に保存され、サーバー不要で動作。
    *   iPhoneの「ホーム画面に追加」に対応（アプリアイコン設定済み）。

## 🚀 機能一覧

### 1. 旅行管理 (Trip Management)
*   **複数旅行の作成・切替**: 複数の旅行プランを個別に保存・管理できます。
*   **複数目的地の設定**: 1つの旅行に対して複数の訪問都市を追加可能。
*   **自動行程生成**: 日付を設定すると、日数分の枠が自動で生成されます。

### 2. 行程表 (Schedule)
*   **タイムライン表示**: 移動、観光、食事などのイベントを時系列で管理。
*   **スポット連携**: 登録済みのスポットから場所を簡単に選択できます。
*   **UI改善**: 入力項目の重なりを防ぎ、タブバーに隠れないようレイアウトを最適化。

### 3. 準備リスト (Packing & Expenses)
*   **持ち物リスト**: 自由にカテゴリーを追加・編集可能。
*   **予算管理 (Walica風)**: 割り勘計算やメンバーごとの支出管理に対応。
*   **カスタマイズ**: 全体・アイテム別のカラータグで視認性を向上。

### 4. スポット (Spots)
*   **旅行別保存**: 登録したスポットは旅行ごとに独立して保存されます。
*   **都市選択**: 予定した目的地（都市）から選択する方式により、入力ミスを防止。

## 🛠 技術スタック

*   **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Animation**: [Framer Motion](https://www.framer.com/motion/)
*   **Icons**: [Lucide React](https://lucide.dev/)
*   **State Management**: Custom `useTripStorage` Hook (旅行ごとの独立した永続化)

## 💻 開発・実行方法

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
# http://localhost:3000 でアクセス
```

## ⚠️ 注意点
このアプリは `localStorage` を使用しているため、データは「ブラウザごと」に保存されます。
PCを変えたり、ブラウザを変えたりするとデータは共有されません。

## 📱 iPhoneでの利用
Safariでアクセスし、共有メニューから「ホーム画面に追加」を選択することで、フルスクリーンのネイティブアプリのような感覚で使用できます。
